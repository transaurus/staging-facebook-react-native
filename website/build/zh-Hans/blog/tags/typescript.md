* [TypeScript declarations shipped with React Native](/zh-Hans/blog/2023/01/03/typescript-first.md#declarations-shipped-with-react-native)
* [React Native documentation is TypeScript First](/zh-Hans/blog/2023/01/03/typescript-first.md#documentation-is-typescript-first)
